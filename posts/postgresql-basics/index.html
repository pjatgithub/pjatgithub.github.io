PostgreSQL Basics

March 25, 2017

Preparation

I have installed the PostgreSQL 9.6.2 on Ubuntu 16.04 LTS, and use command-line tools to manage the server with macOS.

To install PostgreSQL server on Ubuntu 16.04 LTS:

  1. Type sudo apt-get install postgresql to install the server.
  2. Edit /etc/postgresql/9.5/main/postgresql.conf, add the line listen_addresses = '*' to listen on all IP addresses (localhost by default).
  3. Edit /etc/postgresql/9.5/main/pg_hba.conf, and change the line host all all 127.0.0.1/32 md5 to host all all 192.168.1.0/24 md5 which enables remote login from the hosts in the local network.
  4. Run sudo su postgres to switch to the user postgres.
  5. Run psql to connect the PostgreSQL server, and then type \password to change the password of postgres. (After the installation of PostgreSQL, a user and a group both named postgres will be created on Ubuntu. And a user also named postgres will be created on PostgreSQL, like root in Mysql.)

To install Postgres.app on macOS:

  1. Download Postgres.
  2. Open the dmg and drap the app into Applications.
  3. Add /Applications/Postgres.app/Contents/Versions/latest/bin to the PATH variable.

At this time, we can connect PostgreSQL by running psql -h 192.168.1.2 -U postgres in bash.
